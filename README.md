# Aasman_ICTProject

Aasman is a website tracking Pakistan's space missions and honoring Pakistani women scientists reaching beyond the sky, proving the sky is just the beginning, not the limit.

# Technologies Used In Website

HTML : To create Layout of the website. The standard markup language for creating the structure of the website.

CSS : To style the Layout creted from HTML.

JavaScript : To add interactive elements and dynamic features to the website.

# Features of Aasman Website

Following are the features of Aasman Website.

Showcase what Aasman is really about.

Highlight Women in STEM fields across Pakistan.

A wide gallery for Satellites sent to space by Pakistan with the help of SUPARCO.

A form Page specifically designed if you want to be part of our team.
